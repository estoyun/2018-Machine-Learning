### 서버 실행 명령어
```
# 플라스크 웹 서버 폴더로 이동합니다.
cd "Flask Web Server"

# 웹 서버를 실행합니다.
python server.py
```

### 파이썬 데이터 학습 모델 생성 명령어
```
# 파이썬 폴더로 이동합니다.
cd "Python Software"

# 엑셀(Excel) 파일로 학습을 수행합니다.
python offline.py

# (옵션) 학습된 데이터를 확인합니다.
python "predict test.py"
```
